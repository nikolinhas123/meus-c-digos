CHRONOS

A equipe CHRONOS faz parte de uma escola de robótica
Chamada ctrl play que favorece todos os equipamentos para a montagem dos robôs
Para montar os robôs para os combates com diferentes modos
sumô lego: o objetivo deste modo de combate é empurrar o robo adversario para fora da arena
Seguidor de linha: o objetivo deste modo de competitividade é percorrer um percurso em menos tempo que o seu adversario
combate de robôs: o objetivo deste modo e com que os robos lutem para ter a maior quantidade de pontos no final para levar a rodada
